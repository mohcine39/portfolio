Système de Recommandation

Algorithme de filtrage collaboratif pour personnaliser l'expérience e-commerce.

PySpark ALS Flask Redis

Objectif du Projet

Augmenter le panier moyen et le taux de conversion d'un site de e-commerce en proposant aux utilisateurs des articles très pertinents. La recommandation devait se faire en temps réel (< 100ms de latence) lors de la navigation.

Approche & Algorithme

Le système repose sur un Filtrage Collaboratif utilisant l'algorithme des Moindres Carrés Alternés (ALS - Alternating Least Squares) implémenté avec Apache Spark (PySpark MLlib) pour gérer un volume de données massif (Big Data).

L'algorithme décompose la grande matrice "Utilisateurs x Articles" en deux matrices plus petites représentant les caractéristiques latentes des utilisateurs et des produits.

from pyspark.ml.recommendation import ALS

# Configuration du modèle ALS
als = ALS(
    maxIter=10, 
    regParam=0.01, 
    userCol="userId", 
    itemCol="productId", 
    ratingCol="interactionScore",
    coldStartStrategy="drop"
)

# Entraînement sur le cluster Spark
model = als.fit(training_data)

# Génération de 5 recommandations pour tous les utilisateurs
user_recs = model.recommendForAllUsers(5)

Architecture de Production

Pour assurer une faible latence en production, les recommandations pré-calculées par batch (Spark) sont exportées vers une base de données en mémoire Redis. Une API développée avec Flask interroge Redis pour afficher les produits à l'utilisateur quasi-instantanément.

Impact Business

Le déploiement a été validé par un test A/B sur 30 jours contre l'ancien système basé sur les "Meilleures ventes". Les résultats ont montré :

  • Une augmentation de 8% du panier moyen.
  • Une hausse de 12% du taux de clics (CTR) sur la page d'accueil.
