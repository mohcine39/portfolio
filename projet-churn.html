Prédiction d'Attrition Client (Churn)

Identification des clients à risque à l'aide de modèles d'arbres boostés.

Python XGBoost Scikit-Learn SHAP

Objectif du Projet

Dans l'industrie des télécommunications, l'acquisition d'un nouveau client coûte beaucoup plus cher que la fidélisation d'un client existant. L'objectif de ce projet était de construire un modèle de Machine Learning capable de prédire quels clients sont les plus susceptibles de résilier leur abonnement le mois suivant.

Les Données

Le dataset comprenait environ 100 000 entrées avec des variables telles que :

  • Démographie : Âge, sexe, localisation.
  • Services : Type d'abonnement, services internet, sécurité en ligne.
  • Facturation : Montant mensuel, méthode de paiement, facturation dématérialisée.

Modélisation & Implémentation

Après une phase approfondie d'Analyse Exploratoire des Données (EDA) et de Feature Engineering (encodage, gestion des valeurs manquantes, SMOTE pour le déséquilibre des classes), j'ai opté pour XGBoost en raison de ses performances supérieures sur les données tabulaires.

import xgboost as xgb
from sklearn.model_selection import RandomizedSearchCV

# Définition du modèle
xgb_model = xgb.XGBClassifier(objective='binary:logistic', eval_metric='auc')

# Optimisation des hyperparamètres
param_grid = {
    'max_depth': [3, 4, 5],
    'learning_rate': [0.01, 0.1, 0.2],
    'n_estimators': [100, 200, 300]
}

Résultats et Interprétabilité (SHAP)

Le modèle final a atteint un ROC-AUC de 0.89 et un Recall de 0.82 (priorité à la détection des churners potentiels).

Pour rendre le modèle "boîte noire" explicable aux équipes marketing, j'ai utilisé SHAP values. Cela a permis de déterminer que les contrats mensuels ("Month-to-month") et l'absence de fibre optique étaient les principaux facteurs de désabonnement.
